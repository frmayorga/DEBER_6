**31. Una onda pico de 6 V aparece montada en un voltaje de cd de 8 V. Si el voltaje de cd se reduce a 5 V, ¿qué tan negativa llegaría a ser la onda seno?**

De acuerdo a los voltajes superpuestos para saber cuanto es el voltaje mínimo se debe restar el voltaje CD con el voltaje pico: 

VCD - VP = 5V - 6V = -1V

Siendo el valor mínimo sale -1V.
